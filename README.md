# AppleFrameworks

AppleFrameworks is a simple SwiftUI app that I have build throughout the Sean Allen's SwiftUI Fundamentals course.
Technology used:
- SwiftUI
- LazyGrid
- View Model
- UIKit integration
- Safari Web Services
